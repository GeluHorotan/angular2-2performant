## Getting Started

### Prerequisites

* Angular CLI
   ```sh
  npm install -g @angular/cli
  ```

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GeluHorotan/angular2-2performant
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server by using
   ```js
   ng serve
   ```
